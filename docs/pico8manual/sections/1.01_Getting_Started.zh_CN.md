# 开始使用

## 键盘快捷键

**ALT+ENTER**:  切换全屏  
**ALT+F4**:     快速退出 (Windows)  
**CTRL-Q**:     快速退出 (Mac, Linux)  
**CTRL-R**:     重新加载 / 运行 / 重启卡带  
**CTRL-S**:     快速保存工作中的卡带  
**CTRL-M**:     静音 / 取消静音声音  
**ENTER / P**:  暂停菜单 (运行卡带时)  

**玩家1默认按键**: 方向键 + ZX / NM / CV  
**玩家2默认按键**: SDFE + tab, Q / shift A  

要更改默认按键，请使用 PICO-8 内部的 KEYCONFIG 工具:

```
> KEYCONFIG
```
